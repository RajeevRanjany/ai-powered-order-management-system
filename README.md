# AI-Powered Order Management System

An AI-powered Order Management System designed to streamline order processing, inventory management, and business operations.

## Features

- Order Creation & Tracking
- Inventory Management
- Customer Management
- Analytics Dashboard
- AI-Powered Insights & Recommendations
- REST API Support
- Secure Authentication

## Tech Stack

### Frontend
- React.js
- Tailwind CSS

### Backend
- Node.js
- Express.js

### Database
- PostgreSQL

### AI Integration
- Google Gemini API

## Installation

```bash
git clone https://github.com/RajeevRanjany/ai-powered-order-management-system.git
cd ai-powered-order-management-system
npm install
```

## Run Locally

```bash
npm run dev
```
